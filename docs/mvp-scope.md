# Périmètre MVP

Inclus :
- Application statique complète.
- Mode local jouable.
- Cinq jeux MVP.
- Données multilingues.
- Services Firebase optionnels.
- Mode salle en ligne Firebase pour **Tu préfères ?** : lobby, joueurs synchronisés, lien de salle, vote, révélation et score.
- Documentation et déploiement.

Limites :
- Le mode en ligne nécessite un vrai projet Firebase.
- Les jeux Action ou Vérité, Devine le mot, Qui est l’imposteur et Quiz restent local-only dans cette étape.
- Les règles Firestore doivent être durcies avant production.
- Les contenus peuvent être enrichis après les tests utilisateurs.

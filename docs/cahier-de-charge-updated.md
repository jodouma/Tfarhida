# Cahier des charges actualisé

## Contexte

Le projet transforme l'idée initiale de mini-jeux sociaux en une application moderne, responsive et déployable sur GitHub Pages.

## Périmètre MVP

- Gestion de joueurs locaux.
- Bibliothèque de cinq jeux.
- Scores et résultats.
- Contenu tunisien multilingue.
- Mode en ligne optionnel avec Firebase.
- Documentation et workflow de déploiement.

## Contraintes

- Hébergement statique GitHub Pages.
- Pas de serveur privé.
- Pas de stockage local de mots de passe.
- Secrets exclus du dépôt.

# Règles des jeux

## Qui est l'imposteur ?

Tous les joueurs reçoivent un mot secret sauf un imposteur qui reçoit un indice. Après discussion, le groupe vote. Si l'imposteur est trouvé, les autres gagnent des points. Sinon, l'imposteur gagne.

## Action ou Vérité

Un joueur reçoit une vérité ou un défi. Il peut réussir ou passer. Les contenus restent respectueux et non explicites.

## Tu préfères ?

Deux choix sont affichés. Chaque joueur vote, puis la majorité est révélée.

## Devine le mot

Un joueur doit faire deviner un mot sans utiliser les mots interdits. Le timer rend la manche dynamique.

## Quiz culturel tunisien

Question à choix multiples avec correction et courte explication.

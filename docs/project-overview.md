# Vue d'ensemble du projet

Tfarhida est une application web statique de mini-jeux sociaux tunisiens. Elle vise un MVP professionnel pour PFE/PFA, utilisable immédiatement en mode local et extensible vers un mode en ligne avec Firebase.

Objectifs principaux :
- Proposer une expérience mobile-first, conviviale et visuelle.
- Supporter le tunisien, le français et l'anglais.
- Permettre des parties locales sur un seul appareil.
- Préparer des salles en ligne avec authentification et synchronisation temps réel.
